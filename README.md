### 🚀 Featured DevOps Project (In Progress)
#### **Automated Cloud Web Deployment Pipeline**
*A lightweight, cloud-native project designed to demonstrate automated deployment workflows without relying on heavy local machine resources.*

*   **Objective:** Containerize a web application (combining frontend design and backend logic) and automate its lifecycle.
*   **Architecture & Workflow:**
    1.  **Source:** Application source code hosted on GitHub.
    2.  **CI/CD (GitHub Actions):** Triggered automatically on every `git push`. It automatically checks the code and builds a secure **Docker** container image.
    3.  **Cloud Hosting (AWS):** The cloud infrastructure pulls the latest image and serves it live using **AWS** (Free Tier), ensuring zero local deployment overhead.
*   **Key Value:** Demonstrates industry-standard automated delivery (CI/CD) and cloud management, essential for modern software deployment environments.
